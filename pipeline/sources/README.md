# Pipeline source files

`pipeline.py` expects these files in this directory when recomputing `public/data.json`:

- `work_activities.csv`
- `work_context.csv`
- `software_skills.csv`
- `related_occupations.csv`

The repository already includes a deployable, pre-computed `public/data.json`. Add the raw source exports here only when rerunning the model pipeline.
